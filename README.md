# Deveo UI

Deveo UI consists of the following:

* Core styles
* Grid system
* UI components
* Fonts and icons

To get started you'll just need to run,

    $ npm start

access one of the visual test files with your browser, and

    $ open test/visual/button.html
    $ open test/visual/dropdown.html

start developing.
